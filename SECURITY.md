# Security Policy

Не создавайте публичные issue с реальными `golden_key`, данными автовыдачи, экспортами из `storage/` или полными логами.

Сообщайте об уязвимостях автору приватным сообщением: **@voterol**. Приложите обезличенные шаги воспроизведения, версию Cardinal и версию Python.

При возможной утечке немедленно отзовите или замените затронутый ключ аккаунта.
