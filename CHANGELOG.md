# Changelog

## 1.3.0

- Добавлена команда `/lmsearch название или слово`: поиск по части названия без учёта регистра, inline-меню найденных лотов и полный список прямых ссылок.

## 1.2.0

- Добавлен отдельный корневой `VERSION`; metadata плагина синхронизирована с ним.
- Updater сначала проверяет `VERSION` по immutable commit SHA и скачивает код только для более новой версии из того же commit.
- Автообновления включены по умолчанию для новых установок, сохраняют явный legacy opt-out, работают в фоне и больше не блокируют команды запросом согласия.
- Добавлены однократное startup-уведомление всем авторизованным пользователям и явная авторизация updater callback-ов.
- Первая автопроверка отложена до короткого окна opt-out; частично успешная startup-рассылка продолжается без дублей успешным получателям.
- После обновления рассылается уведомление и сохраняется автоматический перезапуск через час с кнопкой немедленного перезапуска.

## 1.1.0

- Добавлены отключаемые opt-in автообновления и ручная проверка через `/lots_update`.
- Обновления загружаются по точному GitHub commit SHA, валидируются и устанавливаются с резервной копией.
- Добавлены настройки обновлений в главное меню и обязательное согласие при первом использовании.
